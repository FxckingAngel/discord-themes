Contribute a Theme

This repo is open to everyone. If you've built a Discord theme, you're welcome here — the process takes about 5 minutes.

1

Fork the repository

Hit Fork on the GitHub repo. This gives you your own copy to work from.

2

Create your folder

Add a folder named YourName Themes/ at the root. Put your .css files inside, organized into subfolders per theme.

3

Run the theme tool

Use themecmd.cmd to register your metadata. It generates the correct entries in theme-authors.json and validates your structure.

4

Open a pull request

Submit a PR from your fork. The validation bot runs automatically and flags anything that needs fixing. Once merged, future PRs from registered contributors auto-merge.
